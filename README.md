# SENTINEL — Login Anomaly Detection Dataset

Synthetic dataset generated for the research paper:
"SENTINEL: An AI-Native Login Anomaly Detection Platform"

## Dataset Details
- Total records: 4,500
- Normal logins: 4,000 (88.9%)
- Attack logins: 500 (11.1%)
- Features: 9 behavioral and contextual login features

## Files
- step1_generate_dataset.py — Dataset generation script
- login_dataset.csv — Generated dataset
